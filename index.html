Glossary

Install Ratio
Ratio of session with an active experiment to total sessions for each day.
Activation Counts
How many new users were enrolled in the experiment each day.
Failed Activations
Table of reasons why users were not enrolled in an experiment. This can expected conditions such as the channel, date ranges, and other experiment conditions. It can also list unexpected exceptions or install failure running the experiment.
Terminations
Table of users departing an experiment per day. There is also a table of reasons users departed an experiment.
